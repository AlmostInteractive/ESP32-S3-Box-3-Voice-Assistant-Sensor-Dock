# ESP32-S3-Box-3-Voice-Assistant-Sensor-Dock

This config integrates Voice Assistant with the S3 Box 3's sensor dock.

Features:
  - Voice assistant with on-device and in-HA wakeword
  - Touchscreen
  - IR learning
  - IR blasting
  - Temperature sensor
  - Battery sensor
  - Radar presence detection
